DapperContext

DapperContext is a DbContext-like implementation for Dapper, supporting the Unit Of Work pattern and simplifying usage with repository classes and IOC containers.

How do I use this thing?

Wow there, hold your horses! This project is still in an alpha state, and I still have to build a NuGet package for it. Expect this within the next day or so :) When the NuGet package has been published, you can head over to the NuGet Package Manager inside Visual Studio and search for DapperContext, or add it to your project using the Package Manager Console window by typing:

Install-Package DapperContext -IncludePrerelease

NuGet will automatically fetch Dapper as well if you haven't already added this great lightweight ORM library to your project.

More information on usage will follow soon, so stay tuned :)
